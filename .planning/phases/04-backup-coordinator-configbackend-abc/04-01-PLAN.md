## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user filesystem ŌåÆ BackupCoordinator | The coordinator reads the user's live config and writes a `.bak` + sentinel under `$HOME/.codex`. Untrusted: the source file may be missing, corrupt, or unexpectedly large. |
| BackupCoordinator ŌåÆ ConfigBackend | The coordinator receives a `backend` and reads `backend.path`. A malicious/buggy backend could point `.path` anywhere. Mitigated by Paths being frozen (D-22) and resolving only under the injected home. |
